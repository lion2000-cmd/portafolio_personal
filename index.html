PHP

2.- Lo siguiente que debemos hacer es un formulario en el cual trabajaremos. a continuacion se muestra un ejemplo en el cual ponemos en practica nuestros conocimientos de HTML y de CSS



3.- Posteriormente debemos generar una conexion en la cual los datos que se ingresen puedan ser guardados en la base de datos. El codigo deberia quedar como en la siguinete imagen.



4.- Por ultimo debemos generar un codigo el cual envie los datos para que nuestra base de datos los reciva y guarde. a continuacion un ejemplo del codigo.

SQL

SQL es un leguaje gestor para el manejo de informacion de bases de datos relacionales. Nos sirve para generar consultas, ingresar, actualizar y eliminar datos entre otros. En mi experiencia e utilizado sql para el registro de datos en bases de datos como hize mencion en la seccion de PHP.


HTML

HTML es un lenguaje de etiquetas del cual su principal funcion es la creacion de la estructura de una pagina web podemos verlo como los simiento como una casa cuando solo son las paredes, sin pintar, los pisos sin lozeta etc. Una pagina con html se veria asi.


CSS

CSS es un lenguaje de hoja de estilos de cascada y se usa para darle forma y como su nombre lo dice estilos a nuestra pagina html, dicho en palabras mas sencillas es como pintar una casa, ponerle lozeta, agregar adornos, darle una mejor apariencia. Una pagina con css puede llegar a verse asi.
